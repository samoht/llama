\subsection{Before reading further}

Currently this manual assumes a familiarity with
\ahref{http://caml.inria.fr/}{Objective Caml}, which is Llama's
immediate ancestor. Readers lacking this prerequisite (or at least
familiarity with a similar language such as Standard ML or Haskell)
are advised to first study that system, working through the tutorial
and such. It will be time well spent.\footnote{Note that for our present purposes,
the module and object systems need not be studied.
In fact, Llama's feature set has much in common with
Objective Caml's predecessor system, Caml Light (and some of Llama's
source code is derived from that system). However, Llama's syntax is
much closer to Objective Caml.}
